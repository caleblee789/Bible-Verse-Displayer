# Changelog

## 1.2.0

- Make the New Living Translation (NLT) the default bundled translation.
- Expand the default verse list to 483 entries covering 500 quoted Bible verses.
- Add NLT attribution and copyright-limit metadata.
- Improve deck-browser quote display with responsive card styling.
- Improve theme-aware light-mode and night-mode colors.
- Share the same quote-card renderer between the deck-browser display and settings preview.
- Remove unused legacy image resources and add-on web export hooks.
- Add tests for UI rendering helpers, theme colors, color validation, and bundled config counts.
